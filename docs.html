💻 Guía de Desarrollo

🔧 Herramientas Recomendadas

  • VS Code
  • Extensión Live Server
  • Navegador actualizado (Chrome, Edge, Firefox)

✏️ Cómo editar

  1. Abre index.html en tu editor de código.
  2. Modifica textos, colores y animaciones dentro del bloque <style> o usando clases personalizadas.
  3. Reemplaza el video del logo en assets/Logo_Anclora_mejorado.mp4 si deseas otra animación.
💡 TIP: Puedes personalizar la apariencia del mar y olas modificando las URLs SVG dentro de los divs .wave1 y .wave2.

🚀 Publicar en GitHub Pages

  1. Haz push de todos los archivos al repositorio remoto.
  2. Desde GitHub, ve a Settings > Pages y selecciona la rama main y la carpeta / (root).
  3. Tu sitio estará disponible en unos segundos desde https://USUARIO.github.io/NOMBRE-REPO.
